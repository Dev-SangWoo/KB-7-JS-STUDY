# JavaScript `this` 정리

## 기본 개념

- `this`는 실행 컨텍스트(함수 호출 : LE + this)가 생성될 때 함께 결정
- 전역 공간에서의 `this`는 기본적으로 `window`를 가리킨다

---

## 전역 this

```js
var a = 1;
console.log(a); // 1
console.log(window.a); // 1
console.log(this.a); // 1
```

- `var a = 1` : 함수 없이 전역에서 선언했기 때문에 전역 LE에 a 변수 저장
- `console.log(a)` : 스코프 체인에서 a를 검색하다가 전역 LE에 저장된 a에 접근 → 1 출력 (window. 생략된 것으로 생각해도 됨)
- `console.log(window.a)` : 전역 LE에 저장된 a 값을 참조해 1 출력
- `console.log(this.a)` : 전역에서 호출했기 때문에 this = window → 1 출력

---

## delete 연산자

- `delete`는 **프로퍼티를 삭제**하는 연산자
- `var a = 1`로 선언하면 `configurable: false`가 붙어서 `delete a`, `delete window.a` 둘 다 삭제 불가
- `window.a = 1`로 직접 할당하면 `configurable: true`라서 `delete window.a` 삭제 가능

---

## 함수 vs 메서드 this

| 호출 방식      | 구분             | this              |
| -------------- | ---------------- | ----------------- |
| `func()`       | 함수 (점 없음)   | `window`          |
| `obj.method()` | 메서드 (점 있음) | 마지막 점 앞 객체 |

```js
var func = function (x) {
  console.log(this, x);
};

var obj = {
  method: func,
  inner: {
    methodB: function () {
      console.log(this);
    },
  },
};

func(1); // this = window (함수로 호출)
obj.method(2); // this = obj (메서드로 호출)
obj.inner.methodB(); // this = obj.inner (마지막 점 앞)
```

- 함수로 **호출한** this → 무조건 전역(`window`) 참조
- 메서드로 **호출한** this → 마지막 `.` 앞에 명시된 객체

## 예제

### Q1. delete

```js
var a = 1;
window.c = 3;

delete a; // ?
delete window.a; // ?
delete window.c; // ?
```

<details>
<summary>정답 보기</summary>

`delete a` : var 선언 → configurable: false → **false**

`delete window.a` : var 선언 → configurable: false → **false**

`delete window.c` : 직접 할당 → configurable: true → **true**

## </details>

### Q2. 함수 / 메소드

```js
var name = '전역은행';
var balance = 0;

var introduce = function () {
  console.log(this.name + '의 잔액: ' + this.balance);
};

var bank = {
  name: '국민은행',
  balance: 100000,
  showBalance: introduce,
  printInner: function () {
    introduce();
  },
};

// 출력값은?
introduce();
bank.showBalance();
bank.printInner();
```

<details>
<summary>정답 보기</summary>

`introduce()` : 전역 호출 → `전역은행의 잔액: 0`

`bank.showBalance()` : 마지막 점 앞 = bank → `국민은행의 잔액: 100000`

`bank.printInner()` : printInner 내부에서 introduce() 함수로 호출 → `전역은행의 잔액: 0`

</details>

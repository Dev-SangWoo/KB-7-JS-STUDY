1. Variable Shadowing (변수 은닉화)

스코프 체인에서 **안쪽 스코프의 변수가 바깥 변수를 가리는** 현상.

let a = 'global'; // (전역 변수)

function inner() {
  let a = 'inner'; // 바깥의 a를 가림 (shadowing + 지역변수)
  console.log(a); // 'inner'
}

inner();
console.log(a); // 'global' ← 바깥은 멀쩡

바깥 변수가 사라진 게 아니라 가려진 것
안쪽에서는 바깥 변수에 접근할 방법이 없음



 2. 전역변수와 지역변수

전역변수(global variable)** → 함수 밖에서 선언, 어디서든 접근 가능
지역변수(local variable)** → 함수 안에서 선언, 그 함수 안에서만 접근 가능

전역변수가 위험한 이유 :** 누구든 접근하고 덮어쓸 수있어서 팀 프로젝트에서 충돌 발생 가능  
지역변수로 가두는 것**이 안전한 코드의 기본

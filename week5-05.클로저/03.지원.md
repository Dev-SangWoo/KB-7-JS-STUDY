### 접근 권한

- public : 외부에서 접근 가능
- private : 내부에서만 사용, 외부 노출x
- protected

자바스크립트는 클로저를 통해 변수에 접근 권한 제어를 할 수 있다.

→ return을 통해 외부 스코프에서 함수 내부 중 일부 변수에 접근 권한을 부여. (외부에 제공하고자 하는 정보들을 모아서 return하고, 내부에서만 사용할 정보는 return하지 않는 방식)

```jsx
var outer = function () {
  var a = 1;
  var inner = function () {
    return ++a;
  };
  return inner;
};

var outer2 = outer();
console.log(outer2());
console.log(outer2());
```

→ outer 함수에서 inner 함수를 반환함으로써 inner는 publiv member가 되고, outer 함수의 나머지 정보들은 private member가 되었다.

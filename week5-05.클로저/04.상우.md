#### 부분 적용 함수 
n개의 인자를 받는 함수에 미리 m개의 인자만을 넘겨 기억시켯다가, 나중에 (n-m)개의 인자를 넘기면 비로소 원래 함수의 실행 결과를 얻을 수 있게끔 하는 함수
this를 바인딩 해야하는 점을 제외하면 bind메서드의 실행 결과가 바로 부분 적용함수입니다.
	
```js
var add = function () {
  var result = 0;
  for (var i = 0; i < arguments.length; i++) {
    result += arguments[i];
  }
  return result;
};

```



함수 안에 함수 즉 클로저를 활용하는 방법들 중에 부분함수 적용이라고 쓸 수도 있음 
```js
// 예제 5-14 부분 적용 함수 구현(1) - this를 안건들이고 부분함수 적용하는 법 

// 매개변수가 선언 안되어 있더라도 묵시적으로 매개변수가 호출되면 arguments라는 배열에 다 넣어줌
var partial = function () {
  var originalPartialArgs = arguments; //받은 매개변수를 따로 저장
  var func = originalPartialArgs[0];
  if (typeof func !== 'function') {
    throw new Error('첫 번째 인자가 함수가 아닙니다.');
  }
  return function () {
    var partialArgs = Array.prototype.slice.call(originalPartialArgs, 1); // 함수만 짤라서 뒤에 있는 변수만 취득
    var restArgs = Array.prototype.slice.call(arguments); //호출한 (add나 dog 안에 들어가 있는 익명함수)에서 받은 매개변수를 restArgs에 담음
    return func.apply(this, partialArgs.concat(restArgs)); // 두 매개변수를 concat- 합쳐가지고 this는 원래 this 유지한채 func(add 등등)으러 apply 해서 출력해줌
  };
};

var add = function () {
  var result = 0;
  for (var i = 0; i < arguments.length; i++) {
    result += arguments[i];
  }
  return result;
};

var addPartial = partial(add, 1, 2, 3, 4, 5);
console.log(addPartial(6, 7, 8, 9, 10));  // 55

var dog = {
  name: '강아지',
  greet: partial(function (prefix, suffix) {
    return prefix + this.name + suffix;
  }, '왈왈, ')
};

console.log(dog.greet('입니다!'));  // 왈왈, 강아지입니다.

```

### 디바운스란?
짧은 시간 동안 동일한 이벤트가 많이 발생할 경우 이를 전부 처리하지 않고 처음 또는 마지막에 발생한 이벤트에 대해 한번만 처리하는 것으로 (FE에 성능 최적화에 도움을 주는 기능)
scroll, wheel, mousemove, resize 등에 적용하기 좋음 
Lodash등의 라이브러리에서 사용 가능 하지만
최소한의 기능은 간단 구현으로도 가능
```js
// 디바운스 함수 정의
var debounce = function (eventName, func, wait) {
  var timeoutId = null;
  return function (event) {
    var self = this;
    console.log(eventName, 'event 발생');
    clearTimeout(timeoutId);
    timeoutId = setTimeout(func.bind(self, event), wait);
  };
};

// 실제 이벤트 핸들러들
var moveHandler = function (e) {
  console.log('move event 처리');
};

var wheelHandler = function (e) {
  console.log('wheel event 처리');
};

// 디바운스 적용해서 이벤트 등록
document.body.addEventListener('mousemove', debounce('move', moveHandler, 500));
document.body.addEventListener('mousewheel', debounce('wheel', wheelHandler, 700));

```

> debounce 는 결국 최종적으로 한번만 실행되도록하는 함수 

1. 'mousemove'라는 이벤트가 발생하면 -> debounce() 함수에 특정 객체가 첫번쨰 매개변수로 넘겨짐 
2. debounce()에는 그러면 4개의 외부 변수가 들어가 있는것 , event, 'move' moveHandler, 500 
	event -> 이벤트 활성 시 내려지는 객체
	'move' -> debounce의 매개변수로 그냥 문자열
	moveHandler -> debounce의 매개변수로 실제 실행할 함수
	500 -> debouce의 매개변수로 몇초뒤에 실행할지 설정하는 숫자

#### 변수 선언과 데이터 할당

- var a = 'abc';
- 메모리에서 빈 공간을 확보하고 그 공간의 이름을 a라고 지정. 이후 사용자가 a에 접근하고자 하면 a라는 이름을 가진 주소를 검색해서 그곳에 문자열을 할당.
- but 실제로 그 위치에 ‘abc’를 직접 저장X
- 별도의 메모리 공간 확보해서 ‘abc’ 저장 후 그 주소를 변수의 메모리 영역에 저장
- 자유로운 데이터 변환과 효율적인 메모리 관리를 위해 변수와 값을 다른 메모리 영역에 저장
- 기존 데이터(문자열 ‘abc’)에 문자를 추가하거나 제거해서 변환을 하게 되면, 문자열을 새로 만들어 완전 새로운 별도 공간에 조장하고 그 주소를 변수에 연결

#### 기본형 데이터 / 참조형 데이터

**불변값**

- 변수 / 상수 : 변수 영역 메모리로 구분. 한 번 데이터 할당이 이루어 진 공간에 다른 데이터를 재할당할 수 있는지
- 불변성 여부 : 데이터 영역의 메모리로 구분.
- 기본형 데이터 (숫자, 문자열, boolean, null, undefined, Symbol)은 모두 불변값 → 변수에 새로운 값을 대입하거나 더하는 등의 변환을 하더라도, 이미 저장된 기존의 데이터를 바꾸는 것이 아닌 완전히 새로운 데이터를 만드는 것. 한 번 만든 값을 변하지 않는다.

**가변값**

- 참조형 데이터는 기본적으로 가변값.

**변수 복사**

- 기본형 데이터 복사 : 복사할 대상(변수)에서 가리키는 값의 주소를 가져와서 연결한다.
- 참조형 데이터 복사 : 복사할 대상의 값의 주소가 들어있는 주소를 가져와서 연결한다.
- 복사한 변수의 값을 바꾸게 될 경우
    - 기본형은 변수에서 가리키는 주소 자체가 바뀌기 때문에 a !== b 가 된다
    - 참조형은 변수에서 가리키는 값을 담아둔 곳을 가리키는 주소가 들어있는 곳을 한단계 더 거치기 때문에 변수가 가리키는 주소는 동일하다. obj1 === obj2
    - 근데 참조형에서 변수를 바꿀 때 아예 변경된 데이터를 새로 선언하면서 변경하면 주소도 바뀐다.
    - 
    
    ```python
    var 0bj1 = { C: 10, d: 'ddd' };
    var obj2 = obj1;
    
    obj2.c = 20;
    => obj1 === obj2
    
    ---------------------------------------
    var 0bj1 = { C: 10, d: 'ddd' };
    var obj2 = obj1;
    
    obj2 = ( c: 20, d: 'ddd' l;
    => obj1 !== obj2
    ```
